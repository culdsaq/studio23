01 La propuesta

QUÉ RESUELVE ESTE SISTEMA

El desglose técnico de un guion es uno de los procesos más costosos en preproducción. Hoy requiere un Director de Fotografía o un Asistente de Dirección leyendo escena por escena para definir tipo de plano, equipo y tono visual.

Script Intelligence automatiza ese proceso. Un modelo de lenguaje entrenado con criterio cinematográfico analiza el guion y devuelve un desglose estructurado listo para producción — en segundos, no en horas.

El sistema es agnóstico al formato: funciona igual para un spot publicitario de 30 segundos que para un largometraje.

03 Prompt engineering

EL PROMPT QUE LO HACE FUNCIONAR

El corazón del sistema. El system prompt no es genérico — fue diseñado con vocabulario controlado específico de producción audiovisual para forzar outputs consistentes y parseables.

Tres decisiones técnicas clave que determinan la calidad del output:

Role + dominio
El modelo se posiciona como asistente técnico de Studio23, no como IA genérica. El contexto de la productora mejora la relevancia.
Vocabulario controlado
Se le pasan enums explícitos de tipos de plano, equipos y vibes. Esto elimina la variabilidad del output y permite parseo directo.
JSON-only output
Instrucción estricta de no agregar texto, markdown ni preámbulos. El modelo devuelve únicamente el array JSON.
system_prompt.js Prompt
const SYSTEM_PROMPT = ` Sos un asistente técnico de producción audiovisual para Studio23, una productora argentina de alto nivel. Tu tarea: analizar guiones y devolver un desglose técnico de planos cinematográficos. // ── OUTPUT CONSTRAINT ────────────────────────── Respondé ÚNICAMENTE con un JSON array válido. Sin texto adicional. Sin markdown. Sin bloques de código. // ── SCHEMA ────────────────────────────────────── [{ "nro_plano": number, "tipo_de_plano": enum(TIPOS), "descripcion_visual": string, "equipo_sugerido": enum(EQUIPOS), "vibe": enum(VIBES) }] // ── VOCABULARIO CONTROLADO ─────────────────────── TIPOS: Plano Detalle | Primer Plano | Plano Medio | Plano Americano | Plano General | Plano Secuencia | Plano Subjetivo | Cenital | Contrapicado | Picado EQUIPOS: Trípode | Gimbal | Steadicam | Grúa | Dolly | Dron | Cámara en mano | Slider | Monopié VIBES: Cinematográfico | Dinámico | Documental | Editorial | Intimista | Épico | Tenso | Contemplativo | Verité | Onírico // ── INSTRUCCIÓN FINAL ─────────────────────────── Cada escena o momento diferenciado = un plano. Sé preciso y técnico. Solo JSON. `;
